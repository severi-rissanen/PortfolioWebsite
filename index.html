Severi's school project website

This page contains links to some selected school projects that I consider to be relevant to the things that I'm working with currently (data science and machine learning), and which also included an original contribution with the choice of topic and methods used. All of them also had a nice report at the end, and most were done personally by me, apart from the Stackoverflow project.

Bayesian occupancy modeling for the Eurasian Blue Tit in Finland

Using data from the citizen science project by Ebird combined with satellite data, I implemented a fully Bayesian occupancy model to analyze the occupancy of the Eurasian Blue Tit in Finland. The project was done as the final project in the Bayesian Data Analysis course at Aalto University. Link to project report.

Exploratory data analysis for stackoverflow user data

This was a project work done in the Applied Data Analysis course at EPFL, with a team of four people (including me). Our idea was to analyze data from Stackoverflow users, questions and answers, with the overarching goal to look at Stackoverflow as a social network. My contribution was quite spread out over different parts of preprocessing and analysis, but one of my "advanced" contributions was the matrix factorization based clustering with respect to programming language use in section "Stack Overflower in a nutshell". Link to data story website.

Visualizing the distribution of composers in the music played by the Helsinki Philharmonic Orchestra over the years

This was a miniproject done for the Information Visualization course at Aalto University. I tried some different ways to visualize the distribution of composers and music played by the Helsinki Philharmonic Orchestra over its lifetime, and especially the composer diversity. Lots of obsessing about data-ink ratio and things like that. Link to project report.

Using multiple correspondence analysis to analyze forest fires in a Portugese national park

As the final project of the Multivariate Statistical Analysis course at Aalto University, I performed data analysis on a data set on forest fires in a Portugese national park. In addition to basic analysis, I used multiple correspondence analysis for exploring relations between different predictors for forest fires. Link to project report.

Protein folding for a three-dimensional toy model using MCMC based simulated annealing

As the project work for the Computational Physics course at Aalto, I studied a toy model for protein folding, and implemented an MCMC based simulated annealing routine for the folding. The idea for the toy model came from a paper which studied a two-dimensional model. I generalized it to three dimensions and came up with the exact modeling approach. Link to project report.

Simulation of a tunable coupler for implementing two-qubit gates between transmon qubits (Bachelor's thesis)

This was my bachelor's thesis project work, where the topic was to simulate a certain two-qubit gate for quantum computers and investigate how good it is. I think that the things learned while doing it have been useful even with my current studies and work, at least from the scientific computing and writing perspective. Link to the thesis
